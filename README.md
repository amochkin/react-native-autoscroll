# react-native-autoscroll

AutoScroll component for React Native

## Installation

**Using npm:**

```shell
npm install @xorders/react-native-autoscroll --save
```

**Using yarn:**

```shell
yarn add @xorders/react-native-autoscroll
```

## Usage
